# Egy-Cyber-Army-DoS
Egy-Cyber-Army is a very powerful and effective DoS/DDoS written in Python. 


# Installation

On linux:

$ git clone https://github.com/drsinaway/Egy-Cyber-Army-DoS

$ cd Egy-Cyber-Army-DoS

$ pip3 install -r requirements.txt

$ python3 main.py
